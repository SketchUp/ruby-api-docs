Class: Layout::TableRow

Overview

A TableColumn is a single row from a table.

Examples:

bounds = Geom::Bounds2d.new(1, 1, 4, 4)
rows = 4
columns = 4
table = Layout::Table.new(bounds, rows, columns)
rows.times { |index|
  row = table.row(index)
}

Version:

  • LayOut 2018
